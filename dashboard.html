What this page is for

This module is the reliability layer of the site. It is the best place to check whether live assistant access, autonomous discovery, and promoted-candidate flow are all healthy before presenting.

Workflow: run python3 ecmo_seed_ranker.py or your custom input command, then python3 build_dashboard_bundle.py, then either open dashboard.html for static viewing or run python3 research_assistant_server.py for the full live assistant.
